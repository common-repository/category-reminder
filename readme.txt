This plugin requires users to assign a category to each post other than
"Uncategorized".

== Description ==

Category reminder uses javascript to assert that a category has been assigned
to a post before publishing it. If no category is assigned, then it will
prompt the user to assign before publishing.

== Installation ==

Activate the plugin. No other steps required.

== Frequently Asked Questions ==

= Why does uncategorized not show up under categories now? =

This plugin removes the uncategorized category from the category selecting
widget. It does not remove the uncategorized category from the database.

= Are there any options? =

Not yet.
